Detailed Description

Implementation of the code generator that emits C++ from the AST.

Converts TranslationUnit, POU, StructType and EnumType nodes into C++ header and source text. The implementation is intentionally minimal and focuses on readable generated output rather than optimization.

Extract the element type from an STArray template string.

Parses a string like "STArray<MATHPROCESSOR, 1, 2>" and returns "MATHPROCESSOR". Used to resolve the underlying FB type when calling methods on array elements.
