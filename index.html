The undoRT Ecosystem

A fully integrated, open-source stack for next-gen deterministic control.

Core OS

A highly optimized Linux Preempt-RT distribution tailored for multi-platform industrial hardware, ensuring microsecond-level deterministic execution.
Runtime

A custom, high-performance multi-threaded SoftPLC runtime infrastructure designed from scratch to leverage the full power of undoOS.
Fieldbus

Ultra-precise fieldbus management. Launching with native support for EtherCAT Master (powered by IgH technology) running deterministically in user-space.
Compiler

The core translation engine. A powerful transpiler that compiles standard Structured Text (ST) into optimized, native C++ code for execution.
Migration

A suite of migration utilities designed to ingest legacy commercial projects (TwinCAT, CODESYS) and seamlessly port them into the undoRT pipeline.
