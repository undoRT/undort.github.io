Detailed Description

template<typename T>
struct undoCore::VarInOut< T >

Wrapper for VAR_IN_OUT parameters.

Template Parameters
TType of the referenced variable

In IEC 61131-3, VAR_IN_OUT parameters are passed by reference. This template provides a clean wrapper for such parameters.
